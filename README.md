# How to run

```bash
# Clone the repo
$ git clone https://github.com/Bascil/react-material-ui-dashboard.git
# cd to the project directory
$ cd react-material-ui-dashboard
# Install dependancies
$ npm i
# npm start to start the server
$ npm start

```
